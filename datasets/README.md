# 数据集目录

本目录用于存放测试数据集。

## 目录结构

```
datasets/
├── M3FD/
│   ├── ir/          # 红外图像
│   └── vi/          # 可见光图像
├── FMB/
│   ├── ir/
│   └── vi/
└── 自定义数据集/
    ├── ir/
    └── vi/
```

## 自定义数据集

创建自己的数据集:
1. 创建数据集文件夹,如 `MyDataset/`
2. 创建 `ir/` 和 `vi/` 子文件夹
3. 将红外和可见光图像放入对应文件夹
4. 确保同名图像配对

**图像要求:**
- 格式: PNG, JPG, BMP
- 红外和可见光图像必须同名
- 图像尺寸: 任意 (会自动resize)
